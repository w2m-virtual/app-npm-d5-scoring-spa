# app-npm-d5-scoring-spa

W2M Virtual · D5 Vendor Scoring intranet (Angular 21).

Dashboard + supplier scoring list backed by `app-mvn-d5-scoring-api` (`http://localhost:8087`).

## Local

```bash
npm install
npm start          # ng serve --port 4204 --host 0.0.0.0
```

Open <http://localhost:4204>.

## Stack

- Angular 21 standalone components, signals
- Angular Material 21 (amber palette)
- Auto-refresh 15s on dashboard and suppliers list
